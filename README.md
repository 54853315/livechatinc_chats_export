livechatinc_chats_export
========================

使用python获取并导出(txt)LiveChat中的客户邮箱，实现了简单的日期条件，更多参数可以在pycurl部分丰富 。 API接口：http://developers.livechatinc.com/rest-api/#get-chats 


***适用于python2.6 ***

搜索功能功能简介： 
开始-结束日期、 开始页-结束页、 打印结果或保存文件（保存位置可在文件中修改）
